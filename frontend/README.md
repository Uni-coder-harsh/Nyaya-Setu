# Nyaya-Setu (Justice Bridge) 🇮🇳

Nyaya-Setu is an **Autonomous AI Paralegal Agent** designed for the "Next Billion Users" in rural Bharat. It bridges the gap between citizens and the legal system by automating the drafting of formal grievances using **Amazon Bedrock** and **Bilingual Voice AI**.

## 🚀 Features
- [cite_start]**Zero-Typing Interface**: 100% accessible to illiterate users via voice-first interaction[cite: 64, 68].
- [cite_start]**BNS 2023 Grounding**: Powered by **Agentic RAG** to ensure advice is strictly based on the new Bharatiya Nyaya Sanhita[cite: 63, 86].
- [cite_start]**Automated Drafting**: Generates court-ready PDFs from unstructured Hindi voice notes[cite: 45, 166].

## 🛠 Tech Stack
- [cite_start]**Frontend**: Flutter (Dart) [cite: 183, 213]
- [cite_start]**Orchestration**: AWS Lambda (Python) [cite: 188, 211]
- [cite_start]**AI Engine**: Amazon Bedrock (Claude 3.5 Sonnet) [cite: 192, 205]
- [cite_start]**Vector Database**: Amazon OpenSearch (BNS Law Vectors) [cite: 194, 223]
- [cite_start]**Speech/OCR**: Amazon Transcribe, Polly, & Textract [cite: 210]
